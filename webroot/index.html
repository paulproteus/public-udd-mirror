Welcome to a public UDD mirror

This is a public, unofficial mirror of the Ultimate Debian Database.

How to use it

Use these credentials:

  • Host: 
    udd-mirror.debian.net
  • Port: 
    5432
  • Database: 
    udd
  • Username: 
    udd-mirror
  • Password: 
    udd-mirror
psql "postgresql://udd-mirror:udd-mirror@udd-mirror.debian.net/udd"

On older psql versions, you will need to use psql --host=udd-mirror.debian.net --user=udd-mirror udd --password. See above for password. You can store the password in the PGPASSWORD environment variable so you don't have to type it every time.

This service used to be available under a different name, nublic-udd-mirror.xvm.mit.edu, with a different user and password (

public-udd-mirror
).
We decided to simplify the naming into 
udd-mirror
. We plan to maintain the old user/password indefinitely, but they should be considered deprecated.

Help

The best way to report issues is using Github issues.

We also tend to track to-do entries there.

The debian-qa mailing list is the best way to reach the UDD team. You can also find Asheesh and Mattia there, and they don't mind if you discuss problems on the mailing list instead of via personal mail to them. Note that since this mirror is unofficial, it is provided on a best-effort basis and Asheesh is fundamentally responsible for maintaining server uptime, not anyone else on the debian-qa list.
